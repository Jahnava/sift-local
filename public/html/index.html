Directions: showAll() will hide all of the white blocks to reveal the background image clearAll() will show all of the white blocks to hide the background image hide("objectID") will show any individual white block finding the # id of the block show("objectID") will hide any indiviudal white block to reveal the background image For Example: show(145) will display the part of the image behind block 145 hide(145) will hide the image behind the block 145
